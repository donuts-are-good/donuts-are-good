## 2023
2023 is dedicated to one project per week. Most of these are stdlib, vanilla, and put together in the span of a laptop charge or some other difficulty modifier. No reason, just fun. 

## weekly projects

- [gofetch](https://github.com/donuts-are-good/gofetch)  - a light alternative to neofetch

- [webring](https://github.com/donuts-are-good/webring)  - webrings like the 90's in vanilla js

- [git-gone](https://github.com/donuts-are-good/git-gone)  - purge a file from your git history

- [curlpass](https://github.com/curlpass/curlpass.com)  - generate passwords with curl or web

- [wolfrss.sh](https://github.com/donuts-are-good/wolfrss.sh)  - terminal rss reader for my friend wolf

- [minatar](https://github.com/donuts-are-good/minatar)  - minimal base64 gravatar generator 

- [notes](https://github.com/donuts-are-good/notes)  - tag based journal for CLI

- [release.sh](https://github.com/donuts-are-good/release.sh)  - script for multiplatform go builds

- [rsa-helper](https://github.com/donuts-are-good/rsa-helper)  - rsa helper functions for signing and encryption

- [curvy](https://github.com/donuts-are-good/curvy) - ed25519/x25519 key conversion test
